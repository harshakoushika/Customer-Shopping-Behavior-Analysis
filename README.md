# 🛒 Customer Shopping Behavior Analysis (SQL + Python)

## 📌 Project Overview
This project analyzes customer shopping behavior using SQL and Python to extract meaningful insights from retail data. It focuses on understanding purchasing patterns, customer preferences, and business trends.

---

## 🎯 Objectives
- Clean and preprocess raw data  
- Handle missing values effectively  
- Analyze customer behavior using SQL  
- Generate insights on purchasing trends  

---

## 🗂️ Dataset
- File: `customer_shopping_behavior.csv`  
- Includes:
  - Customer demographics (Age, Gender)
  - Product categories
  - Payment methods
  - Ratings  

---

## 🛠️ Technologies Used
- Python (Pandas, NumPy)  
- SQL (MySQL)  
- Jupyter Notebook  

---

## ⚙️ Project Workflow
1. Data Loading using Pandas  
2. Data Cleaning  
3. Feature Engineering (Age Groups using `qcut()`)  
4. SQL Analysis (see `sql_queries.sql`)  
5. Insight Generation  

---

## 📊 Key Insights
- Popular product categories  
- Payment method preferences  
- Customer rating trends  
- Age-based behavior patterns
